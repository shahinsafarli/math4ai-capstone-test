# Math4AI Final Capstone

This repository contains the student-facing materials for the Math4AI final capstone.

## Start Here

- Assignment statement: [deliverables/math4ai_capstone_assignment.tex](deliverables/math4ai_capstone_assignment.tex)
- Starter pack overview: [starter_pack/README.md](starter_pack/README.md)
- Starter checklist: [starter_pack/CHECKLIST.md](starter_pack/CHECKLIST.md)

## Repository Contents

- `deliverables/math4ai_capstone_assignment.tex`: the official capstone handout
- `starter_pack/data/`: fixed datasets and split indices
- `starter_pack/scripts/`: deterministic data-preparation utilities
- `starter_pack/report/template.tex`: optional report template
- `starter_pack/src/`, `starter_pack/figures/`, `starter_pack/results/`, `starter_pack/slides/`: minimal scaffold directories for student work

## Starter-Pack Policy

The starter pack is intentionally minimal. It includes:

- fixed digits features, labels, and split indices
- deterministic split and synthetic-data scripts
- a minimal repository skeleton
- a short README and checklist
- an optional report template

It does not include model code, training-loop scaffolding, optimizer implementations, or hidden solutions.

## Suggested First Steps

1. Read the assignment statement.
2. Review the starter-pack README and checklist.
3. Clone the repository and create your working branches.
4. Inspect the provided data files before writing model code.
